## 🎮 PokedexChallengeについて

AIが生成するポケモンクイズを楽しみながら図鑑を完成させていくiOSアプリです。

### 主な機能
- **AI生成クイズ**: Google Gemini AIによる高品質なクイズ
- **図鑑完成**: クイズに正解してポケモンを捕獲
- **詳細統計**: 成績や捕獲状況の分析

### 対応環境
- iOS 15.0以降
- iPhone・iPad対応
